  • Devloped a dynamic hotel booking website with Angular frontend and Spring Boot REST API backend with microservices architecture where users can browse and/or book from a catalogue of hotels.
  • Deployed SPA on Vercel, microservices on Azure Spring Apps, with Azure SQL as Database.
  • List of Services Implemented:
    • Eureka Server Registry to create a common registry for all the services.
    • API Gateway service to route all the request to the respective services.
    • Booking Service and Hotel Service to handle all the Hotels and Bookings related endpoints.
    • Implemented Spring Security with JWT for data integrity and user privacy in Identity Service. API Gateway Service and also other services communicates with Identity Service for token verification.
  • Customer Experience:
    • Integrated search bar to search hotels by city (eg. Gurugram, Noida or Chandigarh).
    • Streamlined booking experience with options for selecting a from-date and to-date and number of rooms on the hotel details page. Used Reactive Forms to update total amount as the user selects these options.
    • Booking can then be completed by making a payment on the checkout page.
    • Implemented Bookings page for tracking previous Bookings.
  • Admin facing side:
    • Facility to add/delete/update any Hotel.
    • All Bookings page that displays a clickable list of all the bookings placed on the website and total Earnings.
    • Booking Details page that lists all the products and their quantity in the order.
  • Redeployed the application as a monolith due to prohibitive cost of deploying microservices.
